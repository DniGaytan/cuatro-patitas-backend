# Welcome to cuatro-patitas backend
## Start up
Node.js is being used to execute the current server, to run it locally, ensure that all dependencies are already installed.

`npm install`

start up the server by running:

`node index.js`

## Test environment

Jest is being used for testing all features supported.

to test all test suites, just hit:

`npm test`

to test an specific test suite, run the following command:

`npm test -t "<Test Suite> <Test>"`
